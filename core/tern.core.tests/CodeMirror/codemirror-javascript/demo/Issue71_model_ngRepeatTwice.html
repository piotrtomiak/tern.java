Tern Demo

Demonstrates integration of Tern and CodeMirror. The following keys are bound:

Ctrl-Space
Autocomplete
Ctrl-I
Find type at cursor
Alt-.
Jump to definition (Alt-, to jump back)
Ctrl-Q
Rename variable

Documentation is sparse for now. See the top of the script for a rough API overview.
